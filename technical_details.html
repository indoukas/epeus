Technical details

The text used in the project was drawn from the TLG E disk (2000).1 It was proofed and customised to reflect recent editorial adjustments.2 The resource was encoded in HTML with a CSS stylesheet. JavaScript was used for the functions of scrolling animation and for extracting, highlighting, bringing into view or clearing the commentary entries.

Citations are appropriately encoded to link to the text in the CapiTainS environment. This is used to access the versions of the text in Perseus Digital Library and permits one to bypass the default settings of the database and to manipulate each citation, so that it includes as much context as needed in any given circumstance.3

The map is imported from Pelagios.4 The map function was coded in JavaScript. Each location variable holds the following values: latitude, longitude, a label with the place-name, my commentary text and a link to the corresponding lemma at Pleiades.

The resource has been tested in current releases of Mozilla Firefox and Google Chrome. In its published version, it will be available online with a Creative Commons BY-NC-SA 4.0 license.
